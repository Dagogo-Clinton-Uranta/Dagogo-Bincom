Pro Shop(Tech Store)

A Fully Functional E-commerce store

Design Insights

  • This is a fully functioning e-commerce store for the sale of electronic goods. It uses a very minimalistic navy and white design. The home page of the site consists of a product carousel of top products, directly below a search bar and search button.


  • The home screen and product details are visible to all site visitors, but to commence the purchase process , one must be signed up to the site. Once signed in , users are able to view their cart , proceed to checkout and pay via PayPal.


  • There is also an admin login screen with extra features for shop owners. Extra features for admins include screens for fulfilling orders, tracking inventory and sending messages to regular users. Click the image/button below to view the full website.

Technical Summary

  • Fundamental technologies used include react, redux ,json web tokens, mongoose and MongoDB.


  • Each screen that the end user sees is a react component ,linked via the react router package. Redux is used here to maintain state across components(global level state) and ensure that all changes to database information have a defined flow through the actions to the reducers and finally the store.


  • Components such as the golden stars used for rating maintain local state as the rating information is particular to that component.


  • The backend makes use of authorisation middleware which calls on the json web tokens library to verify and authorise users when they hit certain endpoints. Upon user sign up their passwords are hashed using the brcypt package and their data is pushed to the mongoDB database via the object relational mapper mongoose.


  • Mongoose is also responsible for creating schemas that direct how the database will accept data for new orders, users and products.


  • Finally, the payments were taken care of by including a PayPal SDK script on the page where the user makes payments and fetching PayPal client ID for the recipient from the backend.
